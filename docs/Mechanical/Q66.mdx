无线模式

| 组合键      | 输出功能     | 说明                        |
| -------- | -------- | ------------------------- |
| FN+Z     | 蓝牙设备1的连接 | 1.长按进入重新匹配 2.短按切换蓝牙设备1    |
| FN+X     | 蓝牙设备2的连接 | 1.长按进入重新匹配 2.短按切换蓝牙设备2    |
| FN+Space | 切换至USB连接 | 1.长按进入重新匹配 2.短按切换到USB连接模式 |



电量和休眠

| 组合键     | 输出功能            | 说明                                                                                                                                                                                                |
| ------- | --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| FN+CAPS | 开启深度休眠          | 在蓝牙模式下且无USB连接，长按组合键3秒，键盘进入深度休眠，其他按键均无法唤醒，再长按FN+CAPS 3秒唤醒键盘，或者插USB电源唤醒键盘;USB连接模式下无作用，(主要作用关闭键盘状态，方便运输)。 关机时，CAPS指示蓝灯闪一下，约0.3秒；开机时，CAPS指示红灯闪一下，约0.3S（仅在蓝牙开关机时指示，USB上电断电时不指示） 注意，如改键，开机时必须按原Fn+CAPS键 |
| Space+Z | 切换为键盘无操作10分钟后休眠 | USB连接状态下，长按3秒启用该功能，出厂设置默认键盘无操作10分钟后休眠。                                                                                                                                                            |
| Space+X | 切换为键盘无操作60分钟后休眠 | USB连接状态下，长按3秒启用该功能                                                                                                                                                                                |
| Space+M | 充电指示灯开启/关闭切换    | USB连接状态下，长按3秒启用该功能                                                                                                                                                                                |

键盘指示灯

| 功能      | 说明                             |
| ------- | ------------------------------ |
| CAPS指示灯 | 指示灯红色；蓝牙状态休眠，指示灯灭，唤醒后，恢复正常指示状态 |
| 低电量指示   | 当电池电量低时，红色灯闪烁                  |
| 充电指示    | 充电绿色灯闪烁指示，充满无指示                |





**电气参数：**

背光全亮工作电流: ≤500mA

背光灯灭工作电流: ≤50mA

休眠电流: ≤1mA

蓝牙模式工作电流: ≤20mA

工作电压: 5±0.5V

电池规格: 锂离子充电电池

电池充电电流: 不大于500mA

**安装教程：**

注：因PCB开槽原因，锁上外壳后再安装轴体, 中间区域轴体可能会无法安装到底，用户可先将中间区域的轴插上后再锁上底壳。
