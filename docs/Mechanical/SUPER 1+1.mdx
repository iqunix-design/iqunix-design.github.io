# SUPER 1+1&#x20;

**注意：本页面仅适用于 SUPER 1+1**

**SUPER 1+1 无开关版-初始激活**

方式一：使用数据线将键盘连接到您的电脑,键盘将自动激\
活并进入有线模式。

方式二:长按FN+ESC组合键5秒,键盘将自动激活并进\
入2.4G无线模式。

![](https://support-assets.iqunix.com/1%2B1-02.png)



**SUPER 1+1 有开关版**

通过键盘顶侧面的实体开关,可进行有线、无线间的切换。

![](https://support-assets.iqunix.com/1%2B1-01.jpeg)



### **如何蓝牙连接**

1. **组合键**`FN + 1`**短按一下，再长按5秒；**

2. **当键盘指示灯快速闪烁时，键盘进入蓝牙配对模式；**

3. **在蓝牙列表中选择 **`IQUNIX 1+1 BT1`** ，指示灯灭，连接即成功。**\
   \*如需连接第2、3台设备，只需将上述组合键中的\[ 1 ]键替换为\[ 2 ]、\[ 3 ]键即可



### **在已配对的设备之间切换：**

**直接按下**`FN + [频道号码]`**（例如，**`FN + 2`**）即可切换到其他已配对的设备。**\
\*各频道组合键对应的数字键, 蓝牙: `[1]、[2]、[3]`, 2.4GHz:`[4]`

### **如何进行2.4GHz连接**

1. **组合键**`FN + 4`**短按一下，将键盘切换到2.4G模式。**

2. **将提供的2.4G接收器插入到设备的USB端口中。**

3. **键盘将自动连接。**\
   \*2.4GHz频道下的指示灯颜色为：粉色。 慢闪为回连状态，快闪为配对状态。
