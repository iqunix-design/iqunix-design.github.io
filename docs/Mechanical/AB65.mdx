# AB65

**注意：此页面内容仅适用AB65-三模RGB版**



**功能组合键**

| 组合键      | 输出功能         | 说明   |
| -------- | ------------ | ---- |
| Fn+Esc   | \~\`         | 短按   |
| Fn + 1   | F1           | 短按   |
| Fn + 2   | F2           | 短按   |
| Fn + 3   | F3           | 短按   |
| Fn + 4   | F4           | 短按   |
| Fn + 5   | F5           | 短按   |
| Fn + 6   | F6           | 短按   |
| Fn + 7   | F7           | 短按   |
| Fn + 8   | F8           | 短按   |
| Fn + 9   | F9           | 短按   |
| Fn + 0   | F10          | 短按   |
| Fn + -\_ | F11          | 短按   |
| Fn + =+  | F12          | 短按   |
| Fn + ←   | 上一首          | 短按   |
| Fn + →   | 下一首          | 短按   |
| Fn + Tab | Win/Mac 布局切换 | 长按5s |
| Fn + Win | Win键 锁定/解锁   | 长按5s |
| Fn + 空格键 | 恢复出厂设置       | 长按5s |



无线模式

| 组合键  | 输出功能   | 说明                       |
| ---- | ------ | ------------------------ |
| Fn+Q | 蓝牙1    | 1.长按进入重新匹配   2.短按切换无线模式  |
| Fn+W | 蓝牙2    | 1.长按进入重新匹配   2.短按切换无线模式  |
| Fn+E | 蓝牙3    | 1.长按进入重新匹配   2.短按切换无线模式  |
| Fn+R | 2.4GHz | 1.长按进入重新匹配   2.短按切换无线模式  |
| FN+B | 查看电量   | 无线模式下查看电量，指示灯每闪烁一次为10%电量 |



灯光设置

| 组合键          | 输出功能       | 说明 |
| ------------ | ---------- | -- |
| Fn + Z       | 背光 开启/关闭   | 短按 |
| Fn + X       | 背光亮度-      | 短按 |
| Fn + C       | 背光亮度+      | 短按 |
| Fn + V       | 光效速度-      | 短按 |
| Fn + N       | 光效速度+      | 短按 |
| Fn + Enter   | 背光【模式】循环切换 | 短按 |
| Fn + R-Shift | 背光【颜色】循环切换 | 短按 |



- 指示灯位于G键与H键之间。

- Caps Lock开启: 白色灯长亮

- 组合键长按切换时: 白色灯闪烁 3 次后返回上次指示状态

- 恢复出厂设置成功后: 背光和指示灯，白光闪烁 5 次
