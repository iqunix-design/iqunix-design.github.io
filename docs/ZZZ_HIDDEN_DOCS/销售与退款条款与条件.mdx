### 标准退货退款政策

我们尽全力设计和制造产品以满足您的需求，因此我们有信心认为您热爱从 IQUNIX 购买的产品。然而，我们也理解有时侯您可能对产品的预期有所不同。虽然发生这种情况的可能性很小，但如您有如此想法时，请知悉以下与产品退货相关的条款。

在您购买的产品未发货时，您可以在对应产品的订单详情中进行自主退款操作。我们将会在一个工作日内以您的原始付款方式向您退款。

如需退回产品，您仅需在产品交付之日起 14 天内在对应产品的订单详情中进行自助退货操作，或者联系我们的顾客服务团队提出退货申请。之后将未损坏的产品连同原始发票（如有）和原始包装一并退回。如果产品在前述时间期限内退回，我们将对其进行更换或以您的原始付款方式向您退款。请注意，我们只支持国内退款，不支持退款到中国大陆以外地区的账户。我们将在收到退货产品后及时向您退款，但您的银行卡收到退款的具体时间取决于您发卡银行的内部政策和操作惯例。任何与退款将何时退回您账户有关的问题，请联系您的发卡银行。请您注意以下几点：

- 产品只能在购买产品的国家或地区进行退货。

- 以下产品不可退货：下载的电子软件、订购的软件升级计划、IQUNIX 礼品卡及任何 IQUNIX 开发人员联盟产品。

- 退款时不会退还加急发货费用。

- 您同意在退回产品之前，有责任自行擦除被退回产品上由您创建的所有个人文件及数据。如果您希望保留任何由你创建的文件或数据，您应在退回产品之前自行备份。退回产品之后，如有任何文件或数据仍然保留在被退回产品上或从被退回产品上擦除，IQUNIX不承担任何责任。

### IQUNIX ITX 产品的特殊退货政策

IQUNIX ITX系列产品退换货时，产品必须原包装退回，且不得有损伤和使用痕迹。产品退回后将由 IQUNIX 的外部工厂予以检测。检测通过后，您可选择换货或根据您购买时的付款方式退款。如产品以原始包装和未使用状态退回，我们将在收到退货产品后的十个工作日内向您发起退款。

关于如何退回在 IQUNIX 购买的产品的完整信息，请访问退货和退款页面。除前述退货权利外，您还享有中国大陆相关三包规定和其他法律项下的权利。IQUNIX 的硬件保证或通过微信联系客服：IQUNIX01 以获取更多信息。

### 订单接受/确认

在订购过程中，我们会通过订单后台与您沟通订单处理和发货准备等情况。但仅当收到您的全额付款时我们才接受您的订单。一旦我们处理了您的订单并准备发货，我们将通过订单后台向您发送发货详情。在多数情况下，有库存的货品将在四十八小时内完成这一流程。货品无库存时，只有在货品到货且准备发货后，您才会收到包含发货详情的信息。请您注意，向您提供发货详情之前，我们向您展示的任何发货或交付时间仅是一个预计的日期。您订单的预计发货时间取决于产品是否有货、付款处理时间和库房处理时间，且不包括在途时间。 IQUNIX可在接受或处理任何订单前要求您提供额外的资格信息。
