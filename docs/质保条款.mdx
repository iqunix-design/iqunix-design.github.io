# 质保条款



**我们的产品享有中华人民共和国消费者权益相关法律法规下涵盖的产品保证。根据*《中华人民共和国消费者权益保护法》*以及有关商品修理、更换和退货的责任规定 (以下简称“三包”)，在特定情况下出现质量问题时，你有权退货、更换或维修。** **自购买之日起（电商渠道按快递签收之日起），产品享一年的质保服务，特殊产品或渠道的质保时间，可依据官方或官方授权渠道的质保政策进行质保。**

### 三包摘要

你根据中国消费者权益相关法律法规享有的三包权利，与你根据IQUNIX授权渠道的质保政策或产品电商页面IQUNIX官方另行提供的质保政策所享有的权利一同存在，而且可以作为该等权利的补充。

由第三方维修或拆动、进水或其他外部原因 (例如，产品被损坏或弯曲) 对产品造成的损坏可能无法修复并不属于适用法律下的保修范围。

在三包规定保修范围内的所有保内维修或更换都将使用全新的零部件或产品。

如果是根据三包规定而进行保内整机更换，更换产品的保修期将从更换之日起重新计算。

- 7天包退：产品自售出之日起7日内发生性能故障，消费者可以选择退货、换货或维修。

- 15天包换：产品自售出之日起15日内发生性能故障，消费者可以选择换货或维修

- 1年包修：产品自售出之日起15日内发生性能故障，消费者可以选择维修

无效购买证明的一些例子是：

### &#x20;产品的有限质保

在有限质保期内，因制造缺陷导致的故障由本公司负责维修或更换。但以下情况属于正常使用过程中的物理损耗或材料老化，不属于产品缺陷，不在免费保修范围内。

如何界定正常使用磨损、正常老化，包括但不限于以下接触类与运动部件：

- 键盘键帽或鼠标按键表面的字符（如丝印、激光蚀刻、热升华）会因长期敲击而逐渐褪色、磨损，键帽表面字符、纹理磨损属于正常磨损。

- 轴体的手感（如段落感、压力克数）会随使用次数增加而轻微变化（例如出现“手感变轻”），或因使用时因环境原因进灰、进水而导致轴体失灵（如无触发、连击）。正常使用下，轴体弹簧轻微疲劳、手感变化属于正常老化及磨损。轴体即使标称敲击使用寿命为若干次，但该结果为实验室方法测试得出，用户实际使用中因环境、操作习惯等因素可能导致实际寿命存在差异。

- 鼠标微动与编码器：微动的单击寿命通常标称为若干万次（该结果为实验室测试数据，实际寿命可能因按压力度、使用频率及环境等因素存在差异）。接近或超过此寿命后，可能出现双击、不灵敏或回弹变弱等现象，均属于正常磨损。编码器（滚轮）的刻度感会因灰尘积聚或机械摩擦而逐渐减弱。

- 拨动开关（如模式切换、DPI切换）：开关的滑动手感会因润滑剂干涸而变得生涩或松动，但若开关无法锁定或接触不良则需判断是否制造缺陷。

- 硅胶脚贴：脚贴在桌面摩擦会逐渐变薄、磨穿。

- 产品外壳涂层：长时间使用后涂层可能发粘、脱落，尤其在湿热、日晒环境下会加速老化。

- 可活动脚撑：反复开合会导致脚撑转轴的阻尼减弱，或脚撑尖端的防滑脚贴磨损。

- 数据线接口：接口内部的金属弹片因频繁插拔而失去弹性，导致接触不良或松动；线材外皮因弯折、拉扯而破损或内部断裂。

如果该产品，含有可充电电池，则不会对电池使用寿命提出任何担保。众所周知，所有可充电电池都会随着时间的推移而丧失一定的充电容量，当然这并不会被认定为产品缺陷。您的电池实际使用寿命取决于您如何使用。
